# **App Name**: BioBloom

## Core Features:

- Custom Link Creation: Enable users to add, edit, and rearrange multiple custom links with titles and thumbnails.
- Profile Customization: Allow users to customize profile details, including bio, profile picture, and display name.
- Themed Backgrounds: Offer a variety of background themes with color variations based on Material You 3.
- Unique URL Generation: Generate a unique, shareable URL for each user's profile.
- AI Theme Suggestion: Suggest themes and color palettes to the user using a generative AI tool, based on keywords the user enters describing the desired aesthetic.

## Style Guidelines:

- Primary color: A vibrant, deep blue (#3F51B5) to convey trustworthiness and stability.
- Background color: A light blue (#E8EAF6) that is subtly desaturated to ensure readability and a calming user experience.
- Accent color: A rich, dark purple (#512DA8) that will contrast to draw attention to CTAs and interactive elements.
- Headline font: 'Poppins', a geometric sans-serif font known for its precision and contemporary style.
- Body font: 'Inter', a grotesque sans-serif known for its modern, objective look.
- Use clean, modern icons to represent links and profile elements.
- Emphasize a clear, user-friendly layout that aligns with Material You 3 principles, utilizing dynamic, yet accessible design components.
- Incorporate subtle, smooth animations for transitions and interactions to improve user engagement.
